time to it. This file is a plain-text record of everyone who has contributed;
the [all-contributors table in the README](README.md#contributors) is the same

## How to get listed

Open a pull request — code, translations, docs, or bug reports all count. Add
yourself to the README table with `npx all-contributors add <your-user> <type>`
(see the [emoji key](https://allcontributors.org/docs/en/emoji-key) for types),
and we'll make sure the co-author credit lands so you show up on the repository's
Contributors graph too.
