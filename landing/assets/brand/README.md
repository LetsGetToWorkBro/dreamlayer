# Brand assets

Drop-in social/profile art in the DreamLayer Platinum style.

| File | Size | Use |
|---|---|---|
| `pfp.png` | 1000x1000 | Profile picture / avatar (X, GitHub, etc.) — the little-Mac mark on the dusk sky; X crops it to a circle. |
| `banner.png` | 1500x500 | X / Twitter header. Wordmark sits in the upper-centre safe zone, clear of the avatar and profile overlay. |

The app icon lives at `host-python/packaging/icon.png` (full) and
`icon_small.png` (the simplified 16/32px mark). The social share card is
`landing/assets/og-card.png`.
